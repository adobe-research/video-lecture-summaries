Scoring function

In short, the score for a set of lines is the sum of the following terms (multiplied by constant coefficients and raised to constant powers):
  • sum of the number of horizontally aligned strokes in each line
  • sum of number of strokes in each line
  • weighted average of the compactness of each line
  • negative of the maximum horizontal gap in each line
  • negative of the maximum vertical gap in each line
  • negative of the overlapping area between lines
For more details and formal description please read the write-up.
