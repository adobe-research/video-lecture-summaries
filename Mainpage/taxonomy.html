Visual objects by temporal property

  • Permanent vs Temporary
  • Appears: Gradually (drawing) vs. Suddenly (e.g. copy and paste, default or starting image)
  • Disappears: Never vs. Gradually vs. Suddenly

Type of Activities

  • Drawing
    • Suddenly appear
    • Gradually draw
  • Animation of object (rotating etc.)
  • Change color of chalk
  • Erase
  • Copy and Paste
  • Scroll
  • Highlighting part of visual
    • Point (hover over element with pointer)
  • Zoom-in, zoom-out
  • Gradual transition between slides

Type of visual objects by content

  • Figures
    • Graph
    • Diagrams
    • Photos
  • Text
    • Equations
    • Handwritten words
    • Typewritten words
  • Annotation
    • Circle
    • Underline
    • arrows
    • braces (curly or square)
    • Cross out (stirke throug)

Relationship between visual objects

  • Correspondence between concepts (e.g. vector vs. scalar)
  • Correspondence between a label and an instance (e.g. the text 'atomic number' and particular numbers '2', '3' etc.)
  • Correspondence between number and count of illustration (e.g. text '6', and six protons drawn)
  • Numbers or variables related through algebraic function
  • Numbers and variables related to non-algebraic function (e.g. atomic number and atomic symbol)
  • Unit conversion
  • Geometric relationship
    • Tangent
    • Coinicide
    • Parallel

Type of Possible User Interactions

  • Links to related topics
    • Internal Links (within the same lecture)
    • Intra Links (within the same series of lecture)
    • External Links (to outside source)
  • Section headings
  • Inserting a quiz (examples)
    • Fill in the blank: blank = part of visual (text or figure)
    • Change or insert specific variable values that have correlated variables or digaram
    • Add similar example problems from the web (ex. Lecture on voltage dividers)
    • Indirect question related to lecture (not part of visual): e.g. What is the definition of...?
  • Additional notes (on the margin): extra explanations, questions
  • Highlighting, emphasizing important parts
  • Adding/modifying(beautifying) text
  • Adding/modifying figure
  • Adding/modifying audio
  • Mark correspondence to other tutorials/lectures on same topic
