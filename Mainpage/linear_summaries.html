Method

Handmade Summary

Rough Segmentation

  1. Segment video in time:
    1. Compute pixel difference between consecutive frames
    2. Diff < Threshold (200) defines draw_start and draw_end time
    3. Diff (frame at draw_start - frame at draw_end) = object
  2. Segment trasncript into sentences with t_start and t_end
  3. Vertially layout objects and sentences according to starting times.

Y-pos clustering

  1. Cluster objects from previous method by y-position of centroid, using meanshift algorithm
  2. Vertically layout clustered object and sentences according to starting time

Finer Segmentation

  1. Remove mouse pointer, count foreground pixel in each frame
  2. Draw start when current foreground pixel > maximum foreground pixel up to now
  3. Draw end when current foreground pixel = previous foreground pixel
  4. Segment using draw start and draw end time

Finer Segmentation +
Center Y-pos clustering

  1. Cluster objects by center y pos using meanshift algorithm
  2. Vertically layout clustered object and sentences according to starting time

Finer Segmentation +
Bottom Y-pos clustering

  1. Cluster objects by bottom y pos using meanshift algorithm
  2. Vertically layout clustered object and sentences according to starting time

Finer Segmentation +
Time clustering

  1. Cluster objects by draw end time using meanshift algorithm
  2. Vertically layout clustered object and sentences according to starting time

Finer Segmentation +
Y-Pos and Time clustering

  1. Cluster objects by draw end time + bottom y pos using meanshift algorithm
  2. Vertically layout clustered object and sentences according to starting time
