1. OpenCV-Python, Check for latest versions
    1. Download and install Python 2.7.x: I installed from windows version from https://www.python.org/downloads/
    2. Download and install Numpy: I installed numpy‑MKL‑1.8.2.win32‑py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/#numpy
    3. Download and install Matplotlib: I installed matplotlib-1.4.0.win32‑py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/#matplotlib
    4. Download and install six: I installed six‑1.8.0.win32‑py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/#six
    5. Download and install python-dateutil: I installed python‑dateutil‑2.2.win32‑py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/#python-dateutil
    6. Donwload and install pyparsing: I installed pyparsing‑2.0.2.win32‑py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/#pyparsing
    7. Download and install scipy: I installed scipy‑0.14.0.win32‑py2.7.exe from http://www.lfd.uci.edu/~gohlke/pythonlibs/#scipy
    8. Download openCV (Check for actual latest version): I installed opencv-2.4.9 for windows, and then copied opencv/build/python/2.7/x86/cv2.pyd
  2. Install scikit
    1. Download and install pip
    2. Add C:\Python27\Scripts to PATH environment variable
    3. Follow rest of instruction in http://scikit-learn.org/stable/install.html
  3. Install PIL: I installed Python Imaging Library 1.1.7 for Python 2.7 from http://www.pythonware.com/products/pil/
  4. To work with mp4 videos: Copy opencv_ffmpeg_64.dll to C:\Python27 directory and rename it to opencv_ffmpeg249_64.dll
  5. Useful Python Extension packages for windows
    1. Download dateutil, pyparsing, six as needed
  6. Install NLTK:
    1. Install setuptool for 64bit python
    2. Follow rest of instruction in http://www.nltk.org/install.html
  7. Install NLTK Data:http://www.nltk.org/data.html
  8. Install sigal for quick html generation:
    C:\Python27\Scripts\easy_install sigal
  9. Install Python IDE: I installed Eclipse with PyDev http://pydev.org/manual_101_install.html
